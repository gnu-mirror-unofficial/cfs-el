Introduction to GNU cfs-el

This is a web page about GNU cfs-el, which is a frontend for using CFS (Criptographic File System - wrote by Matt Blaze - distributed by Debian too) from Emacs.

GNU cfs-el is not part of GNU Emacs.

GNU cfs-el is part of the GNU Project, released under the aegis of GNU.

UPDATE: GNU cfs-el is unmaintained. Please contact the GNU project to apply for maintaining it.

Downloading GNU cfs-el

GNU cfs-el can be found on http://ftp.gnu.org/gnu/cfs-el/ [via HTTP] and ftp://ftp.gnu.org/gnu/cfs-el/ [via FTP]. It can also be found on one of our FTP mirrors.
