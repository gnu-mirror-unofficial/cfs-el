News

  • Changes in GNU cfs-el version 0.5.0 (stable) of the 13th of October 2003
    • Added info manual.
    • Fixes.
  • Changes in GNU cfs-el version 0.4.0 (beta) of the 4th of September 2003
    • Now cfs-el is a GNU program.
    • Added `cfs-make' function (frontend to `cmkdir', to make an encrypted directory).
  • Changes in cfs-el version 0.3.1 (alpha) of the 28th of April 2003
    • Minor fixes.
  • Changes in cfs-el version 0.3.0 (alpha) of the 27th of April 2003
    • Added `cfs-detach-all' function.
  • Changes in cfs-el version 0.2.0 (alpha) of the 20th of April 2003
    • Now it isn't needed to escape special characters in the user input.
    • Now it's possible to choose the name of the cleartext directory.
    • Now supports all the `cattach' options.
    • Now the `cfs-mount' function is more configurable.
    • Added `cfs-list', to list the attached directories.
    • Added selecting if being prompted for setting options.
  • Changes in cfs-el version 0.1.0 (alpha) of the 19th of April 2003
    • First release.
