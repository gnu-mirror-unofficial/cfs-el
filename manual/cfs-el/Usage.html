Usage

Here is a list of the functions defined by cfs-el.

Call them using M-x FUNCTION RET.

cfs-list Function
List attached directories.

cfs-check-if-mounted Function
Check if the CFS is mounted.

cfs-mount Function
Mount the CFS (restart cfsd, the CFS daemon).

cfs-attach Function
Attach an encrypted directory.

cfs-detach Function
Detach an encrypted directory.

cfs-detach-all Function
Detach all the encrypted directories.

cfs-make Function
Make an encrypted directory.

cfs-version Function
Return a string describing the version of cfs-el.
